# ANN Stock Prediction

This repository contains a Jupyter Notebook implementing **Artificial Neural Network (ANN)** for stock price prediction.

## 📘 Project Overview

**something.ipynb** is a machine learning project focused on building and training an Artificial Neural Network to predict stock prices. The notebook demonstrates data preprocessing, model building using neural networks, training, and evaluation.

### Key Features
- Data loading and exploration
- Feature engineering for stock data
- Building and training an ANN model
- Model evaluation and performance metrics
- Visualization of results

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas & NumPy
- Scikit-learn / TensorFlow / Keras (for ANN)
- Matplotlib & Seaborn (for visualization)

## 📂 Files

- `something.ipynb` – Main Jupyter Notebook containing the complete ANN stock prediction workflow
